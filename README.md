# README

* Ruby version
======================
  ruby 3.3.3 

* Database creation:
======================

  run the command
  ```
  rails db:create
  ```
  to create the database

  then
  ```
  rails db:migrate
  ```
  and finally the rake task 
  ```
  rake import:import_csv
  ```
  in order to import from the csv files
